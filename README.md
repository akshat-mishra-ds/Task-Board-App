# Full Stack Task Board Application

This project is a task management application built using React and FastAPI.

### Features:
- Add tasks
- View tasks
- Mark tasks complete
- Delete tasks

### Tech Stack:
- Frontend: React + Axios
- Backend: FastAPI (Python)
- Storage: In-memory list
